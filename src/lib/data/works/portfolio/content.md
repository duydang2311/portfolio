# portfolio

A white-paper, pixelated style portfolio website, built just for me.

## Tech Stack

- **Framework:** [SvelteKit](https://kit.svelte.dev/).
- **Styling:** [Tailwind CSS](https://tailwindcss.com/).
- **Markdown:** [remark](https://github.com/remarkjs/remark), [rehype](https://github.com/remarkjs/remark-rehype).
- **Tooling:** [Vite](https://vite.dev/), [Cloudflare Workers](https://workers.cloudflare.com/).

Happy poking around!
