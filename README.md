# Portfolio

A personal website built to mess around with cool tech.

The looks is heavily inspired from [Commit Mono](https://commitmono.com/).

## Tech Stack

- **Framework:** [SvelteKit](https://kit.svelte.dev/).
- **Styling:** [Tailwind CSS](https://tailwindcss.com/).
- **Deployment:** [Cloudflare Workers](https://workers.cloudflare.com/).
- **Markdown:** [remark](https://github.com/remarkjs/remark), [rehype](https://github.com/remarkjs/remark-rehype), [@shikijs/rehype](https://shiki.matsu.io/packages/rehype).

Happy poking around!
